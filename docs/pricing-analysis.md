# Grimbang 요금제 분석

## 모델 아키텍처 (Tiered Architecture)

| 단계 | 모델 | 단가 | 용도 |
|------|------|------|------|
| 1차 생성 | Imagen 4 Fast | $0.02/장 | 빠르고 저렴한 고품질 생성 |
| 수정/편집 | Gemini 3.1 Flash (2K) | $0.101/장 | 마스크 없는 자연어 기반 편집 |
| 정밀 인페인팅 | fal.ai FLUX.1 [dev] Inpainting | $0.032/장 (1280x720) | 마스크 기반 정밀 수정 (애드온) |

### 모델 선정 근거

- **Imagen 4 Fast**: $0.02 고정요금, 2~4초 생성, 16:9 지원, 포토리얼리즘 우수
- **Gemini 3.1 Flash**: 외부 이미지(Imagen 4 생성본 포함) 편집 가능, 마스크 없이 자연어로 수정, 텍스트 렌더링 정확도 ~90%
- Imagen 4 Fast는 편집/인페인팅 미지원 → 수정 단계에서 Gemini 3.1 Flash 필요

## Precision Edit 애드온 (fal.ai 정밀 인페인팅)

기본 플랜의 수정 3회(Gemini 자연어 편집)와 별개. 마스크 기반 픽셀 단위 정밀 수정.
구독 플랜과 독립적으로 자유 선택 가능.

### 원가

- 모델: fal.ai FLUX.1 [dev] Inpainting
- 과금: $0.035/메가픽셀 (출력 해상도 기준)
- 썸네일 (1280x720 = 0.92MP): **~$0.032/회**

| **Precision 100** | 100회/월 | **$9.99** | $0.10 | $3.20 | **~68%** |

### 기본 플랜 + 애드온 조합 시 마진 개선
